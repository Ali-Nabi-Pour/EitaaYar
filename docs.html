‚ö° Quick Start

Basic usage example:

# Create client with token
client = Client("bot139412:1ef6dbad-0e9f-45f1-8ae0-d8cd3fb6afcf")

# Get bot info
me = client.get_me()
print(f"ü§ñ Bot name: {me.result.first_name}")

# Send message
response = client.send_message(
    chat_id=10872172,
    text="Hello! This is my first message üöÄ",
    title="Test Message"
)

if response.ok:
    print(f"‚úÖ Message sent! ID: {response.message_id}")
else:
    print(f"‚ùå Error: {response.error}")

üîë Authentication

You need an API token from EitaaYar admin panel.

from eitaayar import Client

# Different ways to create client
client1 = Client("your_bot_token")  # Simple
client2 = Client(
    token="your_bot_token",
    base_url="https://eitaayar.ir/api",  # Base URL
    timeout=30,  # Timeout in seconds
    enable_logging=True,  # Enable logging
    log_level="DEBUG"  # Log level
)

üõ†Ô∏è Methods

1. get_me() - Get bot information

# Synchronous
response = client.get_me()

# Asynchronous
response = await client.get_me_async()

2. send_message() - Send text message

response = client.send_message(
    chat_id=10872172,  # Chat ID or username
    text="Message text",
    title="Message title",  # Optional
    disable_notification=1,  # Optional - Silent send
    reply_to_message_id=123,  # Optional - Reply to message
    date=1609459200,  # Optional - Send timestamp
    pin=1,  # Optional - Pin message
    auto_delete_after_views=100  # Optional - Auto delete after views
)

3. send_document() - Send file

# Send file
with open("file.txt", "rb") as f:
    response = client.send_document(
        chat_id=10872172,
        file=f,
        filename="file.txt",
        caption="File description",
        content_type="text/plain"
    )

üö® Error Handling

Error types:

üîç

METHOD_NOT_FOUND

When API method doesn't exist

üîë

INVALID_TOKEN

Invalid or expired token

üí¨

CHAT_NOT_FOUND

Chat or user doesn't exist

‚è∞

TIMEOUT

Connection timeout

Error handling:

try:
    response = client.send_message(...)
    
    if not response.ok:
        print(f"Error: {response.error}")
        print(f"Error type: {response.error_type}")
        print(f"Error code: {response.error_code}")
        
        # Raise exception
        response.raise_for_status()
        
except Exception as e:
    print(f"System error: {e}")
